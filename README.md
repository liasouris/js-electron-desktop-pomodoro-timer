# js-electron-desktop-pomodoro-timer

A simple Pomodoro desktop timer built with **JavaScript** and **Electron**. I built it as a fun project for my own study routine and used it to experiment with color and UI design while keeping the app easy to use.

## Features
- Start, pause, and reset the timer
- Two-session cycle: **Work** and **Break**
- Automatically switches between work and break when the timer ends
- Desktop app built with Electron

## Prerequisites
- **Node.js**
- **npm**

## Install
```bash
npm install
```
